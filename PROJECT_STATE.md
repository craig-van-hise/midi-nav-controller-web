## 3. Current System Capabilities
### Functional Modules
- **Nav Controller**: A 4-way or 8-way directional navigation interface.
- **MIDI Learn Engine**: Allows mapping physical MIDI controls to navigation actions.
- **Context Menu System**: Configurable step sizes and MIDI mappings via right-click.
- **Diagonal Mode**: Toggleable 8-way movement.
